# SparsePortfolioOptimization

This repository contains the code and documentation for the CS 357: Optimization Techniques and Algorithms Project on Sparse Portfolio Optimization.

## Project Description

The goal of this project is to implement and evaluate various optimization techniques and algorithms for solving the sparse portfolio optimization problem. The problem involves selecting a subset of assets from a given portfolio to maximize returns while minimizing risk.

## Getting Started

To get started with this project, follow these steps:

1. Clone the repository: `https://github.com/Krish2208/SparsePortfolioOptimization`
2. Install the required dependencies: `pip install -r requirements.txt`
3. Run the main script: `python data.py`
